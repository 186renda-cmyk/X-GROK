X 内置版

Efficiency First

  • 零延迟响应
    如原生功能般丝滑，无连接等待
  • 上下文感知
    直接读取当前推文、主页或评论
  • 一键翻译
    比传统机翻更地道的口语化翻译

独立 Grok App

Power User

  • 语音模式
    支持实时语音对话与情感交互
  • 深度定制
    更方便地设置 System Prompt (人设)
  • 沉浸体验
    无社交干扰的纯净工作流界面
